##Installation

1) npm install
2) bower install
3) npm start

The UI can be viewed at http://localhost:8000/app

You can change the port number by modifying the 'start' script parameter in the 'package.json' file.
"scripts" {
  "start": "http-server -a localhost -p 9000 -c-1",
}
